Inheritance :
Inheritance like a family tree . Inheritance allows a child class to reuse its parent class code. It can create connections between independent code blocks (classes). This mechanism frees programmers from writing the same code blocks repeatedly. Instead, using inheritance, programmers can reuse some of their code without needing to write it every time.

Polymorphism : 
Polymorphism occurs when a child class modifies or reshapes the inherited properties. Essentially, this means a child class can modify or override some of the properties inherited from its parent.There are two ways to achieve this: method overriding and method overloading:

Abstraction :
Abstraction is the process of hiding complex implementation details and showing only the essential features of an object.We can achieve abstraction using two OOP mechanisms: abstract classes and interfaces.
